# UPM4M (Unity Package Manager for Mac)

## 本アプリケーションはベータ版です。特定条件によっては意図しない動きをする可能性があります。その場合はissueの起票、もしくはお問い合わせをお願いします。ご了承ください。

Unityのアセット（.unitypackageファイル）を整理・管理するためのネイティブ macOS アプリです。

## 機能

### 📦 パッケージ管理
- パッケージの追加・編集・削除・検索
- カテゴリフォルダによるフィルタリング・ソート（名前・追加日・フォルダ）
- サムネイル表示（販売ページのOGP画像を取得・表示）

### 📂 自動ファイル整理
追加時にファイルを整理先フォルダへ自動コピーします。

```
整理先フォルダ/
  カテゴリ名/
    パッケージ名/
      ファイル.unitypackage
```

カテゴリ名やパッケージ名を編集すると、整理先フォルダ内のファイルも自動的に移動されます。

### 🗂️ 追加ファイル・フォルダのコピー
アバターデータのように unitypackage 以外のフォルダやファイルが含まれる場合、追加フォームのトグルをオンにすることで一緒にコピーできます。

### 🔗 販売ページ連携
- URLを登録してページタイトルを取得・保存
- OGP画像をサムネイルとして取得・表示（リスト・詳細パネルの両方に表示）
- ブラウザで販売ページを直接開く

### ✅ 整合性チェック
起動時に整理先フォルダとの差分を自動チェックします。

- 登録済みだがファイルが見つからないパッケージ
- 整理先フォルダにあるが未登録の .unitypackage

差分があった場合は起動時にダイアログで一覧表示されます。各項目からFinderで該当ファイルを直接確認することも可能です。

### ⚙️ 設定
- 整理先フォルダの変更
- カテゴリフォルダの追加・削除
- データ保存場所の確認・Finderで開く
- パッケージ一覧のリセット・完全リセット

## 動作環境

- macOS 15 (Sequoia) 以降
- Xcode 26.2 以降（ビルドする場合）

## インストール

- ここから最新版をダウンロードし、アプリケーションフォルダへ移動してください。

  - https://github.com/NobleSys-tk/UnityPackageManager4Mac/releases

- ソースコードからXcodeでビルドする場合

```bash
git clone https://github.com/NobleSys-tk/UnityPackageManager4Mac.git
cd UnityPackageManager4Mac
open UnityPackageManager4Mac.xcodeproj
```

Xcode で `⌘R` を押してビルド・起動します。

> **注意:** App Sandbox を無効にしています。任意のフォルダへのファイル書き込みを行うための個人ツール向け設定です。

## 初回起動

起動時に整理先フォルダの選択ダイアログが表示されます。unitypackageファイルを整理・保存したいフォルダを指定してください。キャンセルした場合は `~/Downloads/UnityPackages` が自動設定されます。

## 使い方

### パッケージを追加する

1. ツールバーの **「＋ 追加」** をクリック（または `⌘N`）
2. パッケージ名・ファイル・カテゴリ・URLなどを入力
3. URLを入力している場合は **「ページタイトル・サムネイルを取得」** をクリック
4. **「追加」** をクリック → ファイルが整理先へ自動コピーされます

### アバターデータなど複数ファイルを管理する

追加フォームの **「追加ファイル・フォルダも一緒にコピー」** トグルをオンにすると、unitypackage 以外のファイル・フォルダも同じパッケージフォルダへコピーできます。

### パッケージを削除する

削除時は以下の2つから選択できます。

- **リストからのみ削除** — ファイルはそのまま残す
- **ファイルも削除** — 整理先のパッケージフォルダをまるごと削除

## データの保存場所

```
~/Library/Application Support/UnityPackageManager/
  packages.json   # パッケージ一覧
  settings.json   # 設定（カテゴリ・整理先フォルダなど）
```

設定（`⌘,`）の **「データ管理」** タブからFinderで開いたり、リセットができます。

※アンインストールする場合、設定（`⌘,`）の **「データ管理」** タブからリセットを実施してください。
